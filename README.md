# Bookstore Website

A frontend bookstore application built with Next.js that focuses on API integration.
This project implements authentication features such as login and registration,
handles success and error responses, and displays user data from an API.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript
- REST API (Reqres)
- React Icons (Font Awesome)
- Lucide React (icons)

## Features

- User registration (Register)
- User authentication (Login)
- Success and error response handling
- Pigination and display all users from API
- Responsive UI

## API Integration

This project consumes a REST API to handle:

- User data retrieval
- Error and success response management

```bash
npm install
npm run dev
```
